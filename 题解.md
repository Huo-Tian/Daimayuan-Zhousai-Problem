## R1

### A

读入时加个`a[i] % 2 == 1`的if判断,如：
```cpp
if(a[i] % 2 == 1) {
	ma = max(ma, a[i]);
}
```
或者是:
```cpp
ma = (a[i] % 2 == 1 ? max(ma, a[i]) : ma);
```

### B

通过题目，我们可以算出不管怎么放，砖块的面积都是 $$ 2 * 1 = 2 $$

我们在读入n, m后即可直接if判断`n * m % 2 == 0`，如：

```cpp
if(n * m % 2 == 0){
	printf("Yes\n");
} else {
	printf("No\n");
}
```

或者是：

```cpp
cout << (n * m % 2 == 0 ? "Yes" : "No");
```
