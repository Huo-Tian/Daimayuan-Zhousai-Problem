DZP v1.3.4 终末极巅版发布 & v1.3.6 启航公告

亲爱的 DZP 用户与开发者们：

转眼间，v1.3.4 系列即将迎来它的最终章。在过去的一个月里，我们经历了从初版预告到 Snapshot-1/2/3 的全新体验，再到 Snapshot-4/5 的重量级迭代。不知不觉间，DZP 已经陪伴大家走过了很长一段旅程。

今天，v1.3.4 Max_End_Version（终末极巅版）正式发布！面对告别，大家的心情或许各有不同，但无论如何，v1.3.4 都已圆满完成了它的历史使命。接下来，v1.3 家族即将迎来新成员：v1.3.6。

核心更新与修复
作为 v1.3.4 的绝唱，本次发布的 Max_End_Version 带来了以下关键优化：
彻底解决了 v1.2.5 遗留的 R17C 命名问题；
全面重构并优化了 R20 及更早版本的代码结构与美观度。

未来路线图
展望未来，v1.3.6 将是一个全新的起点，其意义不亚于微软用 Windows 11 取代 Windows 10。我们将继续砥砺前行，为大家带来新一代的体验。未来的开发周期规划如下：
v1.3.4 ➔ v1.3.6 ➔ v1.4.0（传说级更新）➔ v1.4.1 ➔ ...

️ 特别说明与致歉
回首来路，从 v0.5.5 预览版到 v1.2.5 的涅槃重生，这一切都源于我最初的小小心愿：为大家提供优质详实的代码。在此，我也要对过去的不足向大家致歉：
因个人原因，自 v1.3.1 起断更了压缩包；
发布了两个准备尚不充分的 Pre-release 版本；
较为仓促地结束 v1.3.4 系列并转向 v1.3.6 的开发。

感谢大家的包容与支持！请注意： v1.3.6 的首个版本（v1.3.6 Snapshot-1）与本次发布的 v1.3.4 Max_End_Version 为同一构建版本，将同步释出。

衷心感谢每一位关注 DZP 的用户与开发者，让我们共同期待 v1.3.6 的新篇章！
